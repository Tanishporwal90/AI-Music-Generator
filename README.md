# 🎵 AI Music Generator

An interactive AI Music Generator built using HTML, CSS, and JavaScript that allows users to generate and play music based on different moods.

## ✨ Features

- 😀 Happy mood music
- 😌 Relax mood music
- 😢 Sad mood music
- 💕 Romantic mood music
- ▶️ Play music
- ⏸️ Pause music
- 🎶 Animated music visualizer
- 🎨 Attractive glassmorphism UI
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
